# RaftKV
> Codename: Flotilla

A production-grade distributed key-value store built with Raft consensus in Go.

## Features

- Strong consistency via Raft consensus
- Crash recovery with Write-Ahead Log (WAL)
- Fast snapshots for bounded recovery time
- Multiple APIs: HTTP REST, gRPC, custom binary protocol
- Production-ready observability (metrics, logs, tracing)
- Horizontal scalability via sharding
